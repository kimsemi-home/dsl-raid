# Diagnostics Panel

Diagnostics panel shows analyzer output and links each diagnostic to all of its
subjects.

Selecting a diagnostic should highlight every related state/transition subject
in the canvas.

Diagnostic rows should include assertion context when available:

- proposition ID, such as `V007`
- assertion ID, such as `assertion:fsm.initial_exactly_one`
- diagnostic code, such as `FSM007`
- severity
- subjects
- evidence
- suggested fix
