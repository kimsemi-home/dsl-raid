# Code of Conduct

DSLRaid is intended to be a thoughtful, technical open source project. Keep
discussion respectful, specific, and focused on making the work better.

## Expected Behavior

- Be clear and kind in reviews.
- Critique designs and code, not people.
- Give contributors enough context to act.
- Respect project boundaries around ADR-required changes.
- Assume good intent, and ask clarifying questions when intent is unclear.

## Unacceptable Behavior

- Harassment or personal attacks
- Public sharing of private or secret project data
- Dismissive or hostile review comments
- Repeatedly bypassing agreed ADR and security processes

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or
pause discussions when needed to keep the project safe and productive.
